NewbornRisk is an advanced web server facilitating Next-Generation Sequnecing (NGS)-based newborn screening by providing functions of:
1) Streamline pipeline for analysing NGS data (WGS, WES, etc) for either a single newborn or batch samples;
2) Customization of suggestions and comments for each newborn;
3) Family Pedigree/Segregation Analyasis;
4) Generating Publication-ready figures.
5) NewbornRisk Database: https://newbornrisk-db.serve.scilifelab.se/app/newbornrisk-db \
It is developed for facilitating the clinical early detection, reporting, and populational genomic research of the neonatal rare diseases.
